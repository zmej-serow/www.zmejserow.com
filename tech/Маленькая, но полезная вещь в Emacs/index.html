Маленькая, но полезная вещь в Emacs
29/08/2020

Бывает, возникает необходимость отредактировать какой-то read-only файл (например, system-wide конфиг). Понятно дело, можно запустить отдельный инстанс редактора из терминала, предварив его командой sudo. В емаксе можно обойтись без этого.

Если перед именем открываемого файла дописать /sudo::, емакс попросит авторизоваться, и после этого откроет файл так, как будто редактор запущен под рутом: без ограничений. Дико удобная штука, кто не знал -- пользуйтесь. Экономит время. :)
